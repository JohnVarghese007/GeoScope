# GeoScope

**Geospatial Data Visualization Platform**

GeoScope is a modern platform for exploring global events, metrics, and geospatial datasets through an interactive 3D Earth interface. It combines real‑time data ingestion, geospatial analytics, and a high‑performance visualization layer to make complex global patterns easy to understand.
