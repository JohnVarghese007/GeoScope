# Frontend Info

This frontend uses **React + Vite** for fast development, hot‑module reloading, and a clean modern toolchain.  
Will probably use Three.js or some other rendering library to render 3D model and such

---

## Development

Install dependencies:

```bash
npm install
```

Run dev server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

## Tentative Project Structure
( this is just the default structure you get when you create react app, frontend can change this to whatever)
```bash
frontend/
│
├── public/                 # Static assets
│
├── src/
│   ├── App.jsx             # Root component
│   ├── main.jsx            # Entry point

│   └── components/         # (Optional) Reusable UI components
│
├── index.html              # HTML template
├── package.json
├── vite.config.js

```
